# Accessibility
This site was built by Brendan S for DH 101.

I have tried to make it accessible by:
1. Implemented visible keyboard focus states with :focus-visible outlines and interactive styling to make navigation easier.
2. Added alt text for artifact and image-related content to describe what is shown.
3. Included a low-motion sliding side navigation bar for quicker, less overwhelming movement through the page.
4. Used readable visual choices with larger line height, clear spacing, and high-contrast styling to improve usability.
5. Designed the website to work across different screen sizes and devices (while noting it may break after future updates).
6. Used accessibility tools and guidelines, including WCAG references, to review and improve the site.
7. WAVE Accessibility Evaluator reported an AIM score of 10 out of 10 for screen-reader presentation.
